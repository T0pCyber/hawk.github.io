About Hawk


How to Get Involved

Hawk is actively seeking new developers to join our open-source community. We welcome developers across different specialties:

Development Contributions

  • PowerShell Developers: Help expand Hawk's core investigation capabilities
  • Front-End Developers: Enhance the Hawk website with modern web technologies
  • Test Engineers: Create and maintain automated tests using the Pester framework

Testing & Validation

  • Test new releases and features
  • Validate investigation workflows
  • Report bugs and issues through GitHub
  • Suggest improvements based on real-world usage

To begin contributing to Hawk, simply clone the Hawk GitHub repository, select an open issue to tackle, and start coding! For any questions on the project structure / code base, contact the Hawk development team via email or via the questions issue form on the Hawk GitHub repository.
